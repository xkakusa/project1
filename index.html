Section 1: Basic of the web

The web

The web is like a bunch of computers talking to each other. Clicking on a link sends http request
to a server and retrieves HTML file and sends it to your web browser to display the page

HTML and Tags

HTML stands for hyper text markup language and is what forms the majority of content on the web.

HTML files are made using elements. HTML tells the browser the type of element by using HTML tags

Some of the tags are

  • "p" for paragraphs
  • "b" for making words bold
  • "em" for emphasizing words
  • "img" for using images
  • "a" for putting links in your page

Inline and block

Inline elements such as span and "a". They are normaly displayed without line breaks.
Block elements such as "p" and "div" start and end with a new line, they also create and invisible box around the content.

The way Computers work

Computers only know how to do exactly what you tell them to do. By leaving on simple instructions such as not putting a closing tag or using the correct tag will cause big problems.

Section 3: CSS

Repetition

Avoiding repetition can reduce the amount of errors by copy and pasting elements, classes, and id's. Using the same code allows you to change different attributes and be more consistant without manually having to change other elements and potentionaly creating a problem for yourself

CSS

CSS or Cascading Style Sheets, is used to give style to HTML to change how the page looks and feels.
